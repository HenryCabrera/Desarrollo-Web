# Página Web Informativa - Excursión a Semuc Champey

Este proyecto es una página web desarrollada en HTML que promociona una excursión al destino turístico Semuc Champey, ubicado en Alta Verapaz, Guatemala. La página incluye una descripción detallada del lugar, galería de imágenes, itinerario de la excursión, actividades disponibles, y otros elementos informativos útiles para los visitantes.

## Primer Sitio Web Publicado
https://sitiowebturistico.netlify.app/

# Actualización:

## Mejoras en la estructura HTML:

-Se reorganizó el contenido con etiquetas semánticas como <header>, <main>, <section>, <nav>, <figure> y <footer> para mejorar la legibilidad y el orden del código.
-Se optimizó la accesibilidad agregando descripciones alternativas en imágenes y roles/atributos apropiados como aria-labelledby y scope.

## Estilos CSS aplicados:

-Se usó una paleta de colores naturales con tonos turquesa y verde (inspirados en Semuc Champey) para transmitir frescura y naturaleza.
-Se implementó un diseño responsivo y moderno, con: navegación centrada y estilizada, imágenes en galería con disposición en cuadrícula flexible (flexbox), tabla de itinerario estilizada con colores suaves, tipografía legible y profesional.
-Efectos visuales al pasar el mouse sobre enlaces (hover).
